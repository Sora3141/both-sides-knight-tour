# Both-Sides Knight's Tour — 両面ナイトツアー

**表と裏の 2 面を持つ盤**でのナイトツアー パズルです。
ナイトは表裏を行き来しながら、両面の全マスをちょうど 1 回ずつ通る経路を探します。
Three.js による 3D 表示で、盤を回転させて両面を確認できます。

## 🔗 リンク

- リポジトリ: https://github.com/Sora3141/both-sides-knight-tour
- デモ: https://sora3141.github.io/both-sides-knight-tour/

## 概要

- 表面・裏面をつないだ盤上でのナイトツアー。裏側への回り込みが攻略の鍵。
- 移動可能なマスをハイライトし、クリック / タップで進む。
- **UNDO**（1 手戻る）・**REBOOT**（やり直し）対応。
- ドラッグで視点回転・ホイール / ピンチでズーム。

## 使用技術

- **Three.js**（3D 描画）
- HTML / CSS / Vanilla JavaScript
